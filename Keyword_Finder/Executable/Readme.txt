이 Keyword_Finder 프로그램은 hwp 파일로부터 키워드들을 추출하는데 도움을 주는 프로그램입니다.

hwp 파일로부터 뽑아온 자료들은 csv파일에 저장이 되며, 이 csv파일을 이용한 word cloud 기능도 포함되어 있습니다.

안에 있는 Keyword_Finder_lib 폴더와 models 폴더, 그리고 hwp 폴더는 Keyword_Finder.exe 원본파일과 반드시 같은 곳에 있어야 합니다.

한글과 영어 모두 monogram과 bigram을 동시에 추출하여 주며 stopwords.txt에는 결과물을 보고 결과에서 제외하고 싶은 단어를 적어주시면 됩니다.

영어의 경우 대소문자 관계없이 모두 프로그램 내부에서 소문자로 처리가 되기 때문에, stopwords.txt에 작성시에 대소문자 구분하실 필요가 없습니다.

이 프로그램은 기본적으로 hwp를 txt로 변환하여 데이터를 추출하기 때문에 ETRI 자체 보안프로그램에 의해 막히는 경우가 있습니다.

이 경우, 프로그램이 멈추게 되므로 바로 프로그램 종료 후, 해당 파일을 삭제하고 다시 실행시켜주시면 됩니다.

자체 중복체크 기능이 있기 때문에 이미 csv나 word cloud 파일이 만들어져 있을 경우, 해당 파일은 프로그램 재시작할 때 분석 대상에서 제외됩니다.

만약 stopwords를 새롭게 추가한 결과물을 원하신다면 해당하는 csv파일과 word cloud 파일을 지워주시고 실행시켜주시면 됩니다.
